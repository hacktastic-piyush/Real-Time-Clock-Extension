# Real-Time-Clock-Extension
An extension that displays a real-time clock in a pop-up.
<p>You can download the file and try using this extension on your chrome browser</p>
<h4>Steps to use the Extension</h4>
<ol>
  <li>Create a folder and add all the files in the created folder ( all files should be at same location )</li>
  <li>Go to Chrome -> Extension -> Manage Extension</li>
  <li>In Manage Extension, On the developers mode</li>
  <li>Now a Load Unpacked Button will appear on top, click and browse to your folder and load the extension</li>
  <li> Your Extension is ready to be used.</li>
</ol>
